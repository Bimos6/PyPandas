# Обработка данных владельцев компаний

Скрипт для очистки, нормализации и анализа данных о владельцах компаний.

## Функционал

**Очистка и нормализация:**
- ФИО → формат "Фамилия И.О."
- Названия компаний → без кавычек и лишних пробелов
- ИНН → строковый тип
- Доли владения → проценты (0.5 → 50%)
- Даты → формат YYYY-MM-DD

**Анализ:**
- Поиск записей без ИНН
- Поиск пустых долей
- Компании с долями >100%
- Компании с изменениями долей
- Владельцы в нескольких компаниях

## Файлы

- `data.xlsx` - входные данные (все поля в одной колонке)
- `result_data.xlsx` - результаты обработки

## Листы в result_data.xlsx

|         Лист           |              Описание             |
|------------------------|-----------------------------------|
| Обработанные данные    | Все записи после нормализации     |
| Статистика             | Общие показатели                  |
| Доли более 100%        | Компании с суммой долей >100%     |
| Изменения долей        | Компании с изменениями во времени |
| Владельцы в нескольких | Владельцы > 1 компании            |
