# Тестовое задание: Обработка данных владельцев компаний

## Описание
Скрипт для очистки, нормализации и анализа данных о владельцах компаний.

## Функционал
- Нормализация ФИО к формату "Фамилия И.О."
- Очистка названий компаний от кавычек
- Приведение ИНН к строковому типу
- Поиск пропущенных значений
- Конвертация долей владения в проценты
- Нормализация дат к формату YYYY-MM-DD
- Анализ компаний с долями >100%
- Поиск изменений долей во времени
- Поиск владельцев в нескольких компаниях

## Установка и запуск
```bash
pip install -r requirements.txt
python main.py
